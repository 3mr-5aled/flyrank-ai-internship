# Flyrank AI Internship Assignments

This repository contains assignment work for the Flyrank AI internship, organized into two tracks:

- `AI Fluency`
- `Backend AI Engineering`

Each track is split into weekly folders so the work stays easy to review and compare over time. Week 1 currently contains written exercises, prompts, summaries, and supporting screenshots.

## Repository Layout

```text
AI Fluency/
  week-01/
  week-02/
  week-03/
  week-04/
Backend AI Engineering/
  week-01/
  week-02/
  week-03/
  week-04/
```

## Working In This Repo

- Keep each week’s deliverables inside the matching week folder.
- Store supporting images in the relevant `screenshots/` folder.
- Use Markdown for written responses so the files stay easy to diff and review.

## Notes

This repository is documentation-first, so there is no application build step or runtime setup required.
