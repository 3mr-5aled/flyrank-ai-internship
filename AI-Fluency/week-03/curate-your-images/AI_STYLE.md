# AI Style Guide

AI-generated visuals are limited to decorative assets only.

## Style

- Modern SaaS
- Minimal
- Soft gradients
- Blue
- Indigo
- Purple
- Rounded geometry
- Clean whitespace
- Flat illustrations
- No text
- No people
- No fake interfaces
- No mock devices

## Example Prompt

Create a modern abstract background using blue, indigo and purple gradients with smooth geometric mesh patterns, soft lighting, rounded shapes and generous whitespace. Flat vector style. No people. No text. Minimal SaaS aesthetic.

The same prompt should be reused with minor adjustments to maintain visual consistency across every generated asset.
